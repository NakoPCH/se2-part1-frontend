 # Frontend Web Application – SE2

Το παρόν repository περιέχει τον frontend κώδικα της εφαρμογής που αναπτύχθηκε για το μάθημα **Μηχανική Λογισμικού 2 (2025–2026)**.  
Το frontend υλοποιεί τις βασικές οθόνες της εφαρμογής και επικοινωνεί με τα διαθέσιμα endpoints του backend.

---

## Λειτουργίες Frontend
- Περιλαμβάνει **4+ οθόνες** βασισμένες στα mockups της ΜΛ1.
- Πραγματοποιεί **τουλάχιστον 5 API calls** προς τον backend.
- Εμφανίζει δεδομένα που παρέχονται από τον backend μέσω mock/hardcoded πληροφορίας.
